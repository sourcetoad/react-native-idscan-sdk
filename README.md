# react-native-idscan-sdk

React Native ID Scanner wrapper for idscan sdk

## Installation

```sh
npm install react-native-idscan-sdk
```

## Usage

```js
import { multiply } from "react-native-idscan-sdk";

// ...

const result = await multiply(3, 7);
```
